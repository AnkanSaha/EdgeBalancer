# Code of Conduct

## Our Standard

Contributors are expected to be respectful, direct, and collaborative.

Examples of expected behavior:
- giving actionable technical feedback
- assuming good intent
- discussing disagreements with evidence
- keeping review comments focused on the work

Examples of unacceptable behavior:
- harassment or personal attacks
- discriminatory language
- intentionally disruptive behavior
- publishing private information without consent

## Enforcement

Project maintainers may remove comments, reject contributions, or restrict access for behavior that violates this code of conduct.
