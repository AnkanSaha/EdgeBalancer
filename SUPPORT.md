## Good Issue Reports Include
- the exact page or API route involved
- reproduction steps
- expected result
- actual result
- relevant logs or screenshots
- whether the issue affects create, update, delete, rollback, or routing

## Configuration Help
Most setup issues come from:
- missing Cloudflare token permissions
- invalid `ENCRYPTION_KEY`
- missing MongoDB connection values
- hostname conflicts with an existing Worker domain
